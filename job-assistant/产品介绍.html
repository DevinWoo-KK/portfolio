技术亮点

极致成本控制

简历文本固化在 Prompt 前缀，利用 DeepSeek Prompt Cache 机制，每次评分仅计岗位内容 Token。选用 Flash 模型处理高频评分，日均 API 开销不到五毛钱。

健壮反爬体系

Firefox Cookie 注入绕过登录验证码；BOSS PUA 字体 Unicode 解码；三平台独立 Playwright 脚本错峰执行；验证码自动检测 + Telegram 即时告警。

双维度 AI 评分

不只做关键词匹配——从人类招聘视角评估薪资合理性、公司靠谱度、行业匹配度、加班信号、外包嫌疑，避免"关键词全中但实际是坑"的误投。

全闭环自动化

Cookie 检查 → 扫描 → 评分 → 打招呼 → 回复监控 → 发简历 → Excel 追踪 → 日报，18 个 Cron 任务协同，零人工干预。
